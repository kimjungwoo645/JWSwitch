# JWSwitch 설명서

## 기능
#### 1.스위치 핸들 이동
-스위치1은 누르면 한번에 이동
-스위치2는 누르고 다시 누르면 제자리로 복귀

#### 2.스위치 작동시 배경색 변경
-버튼이 On,Off 될시 배경색이 서서히 변함
